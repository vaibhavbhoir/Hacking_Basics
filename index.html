Virtualization in Simple Terms

Virtualization is when we run an operating system on top of another operating system.
Take a big computer with lot of ram ,storage, processing and network capacity and use hypervisor software to run virtual machines on it.

Benefits of virtualization:
  • Reduced costs
  • Upgrade with mouse clicks
  • Minimum downtime
  • Incresased IT productivity
  • More efficient
Install your first virtual machine
  • Install virtual box
  • Install unbuntu linux iso file from internet
  • Open virtual box and create new machine
  • Enter name select type linux version unbuntu x64
  • Enter ram
  • Choose dynamically allocated and choose amount of space
  • Choose iso file
  • Install linux make user and update the system. Ready to use
